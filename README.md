AI Finance Tracker
  - A full-stack application that tracks expenses and uses a custom Linear Regression algorithmn to predict future financial trends.

Tech Stack:
  Frontend: React.js, CSS, Fetch API
  Backend: Java, Spring Boot (REST API)
  Algorithm: Least Squares Method

Features:
  Transaction Logging: Add and view expenses in real-time.
  AI Predictions: Automatically calculates projected spending for the upcoming month based on history.
  Full-Stack Sync: React Frontend updates instantly when the Java Backend processes data.
  Custom Math Engine: Linear Regression logic implemented from scratch in Java.
